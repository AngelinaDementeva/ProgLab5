Папка OrganizationWork - проект, разработанный средствами программы NetBeans 8.2 на языке программирования Java 8.
В папке с проектом содержатся:
Файл dataSet.csv - файл с сохраненными организациями. Данный файл применяется для запуска программы.
Файл script.txt - файл, содержащий команды для выполнения. Необходим для работы команды "execute_script".

Папка OrganizationWork_javadoc - содержит разработанную документацию к проекту. Для запуска документации необходимо запустить файл index.html.

Для работы программы нужно в среде разработки указать правильные аргументы командной строки - 
нужно указать путь к файлу с организациями. Сейчас там указан путь "I:\dataSet.csv", Вам необходимо указать свой путь,
путь, по которому у Вас будет распологаться файл "dataSet.csv".

!!!ВАЖНО!!!
Чтобы не работать через аргументы командной строки просто закомментируйте строку organizationWork.run(); (строка 23) и
раскомментируйте строку organizationWork1.run(); (строка 29) - путь уже указан (так как файл dataSet.csv находится в корне проекта,
то программа автоматически заппустится с этим файлом).
